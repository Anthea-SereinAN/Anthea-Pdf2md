MD转换器

PDF / Word / PPT → Markdown

将多种格式的文档高效转换为 Markdown，适配 AI 更擅长读取 Markdown 的场景。 采用双管线架构自动区分文字型与扫描版 PDF，支持 OCR 识别与 AI 校对， 提供可视化 Web 界面，可打包分发独立使用。

✦ 核心功能

🔀
双管线架构
自动检测 PDF 类型：文字型直接提取文本，扫描版走 OCR 识别，无需手动切换
⚡
多进程并行
利用多核 CPU 并行处理页面，大幅缩短长文档转换时间
🤖
AI 校对
转换后可调用 AI 对结果进行校对和格式优化，提升 Markdown 输出质量
🌐
Web 可视化界面
Flask 驱动的 Web 界面，上传→转换→预览→下载全流程可视化操作
📦
打包分发
使用 PyInstaller 打包为独立可执行文件，无需 Python 环境即可运行
📄
多格式支持
支持 PDF、Word (.docx)、PPT (.pptx) 等常见文档格式，统一输出标准 Markdown
